The Ada compiler on sleipnir, gnu gnat, supports the Ada95 standard. GNAT is a front-end to GNU gcc. Read this before starting. 

Filenames:

  test.adb   // ada package body - source code
  test.ads   // ada package specification - like a header or class definition
  test.ali   // created by compiler for the linker - you can delete this file
  test.o     // object file created for gdb debugger by compiler - can delete
 
Compile and link the short way:
 
 $ gnatmake test.adb 

OR compile and link the long way: 

 $ gcc -c test.adb
 $ gnatbind test
 $ gnatlink test

Run the executable

  $ ./test
